api documentation

biz logic URL method query parameters response
allows a user to be created /users POST none JSON of new user. includes id, username, firstname, lastname, and favorite movies
allows a movie to be created /movies POST none JSON of new movie. includes id, name, description, genre, and director
allows a director to be created /directors POST none JSON of new director. includes id, name, bio, and birthyear
allows a genre to be created /genres POST none JSON of new genre. includes id, name, and description
allows a user to be deleted /users/:Username DELETE :Username confirms user deletion
allows a user to be updated /users/:Username PUT :Username confirms user information update
add movie to favorites /users/:Username/movies/:MovieID PUT :Username/movies/:MovieID adds movie to favorites
returns a list of all movies /movies GET none JSON object of all the movies
returns movie name /movies/:id GET :id JSON object the movie
returns a list of all directors /directors GET none JSON object of all the directors
returns director name /directors/:id GET :id JSON object the director
returns a list of all genres /genres GET none JSON object of all the genres
returns genre name /genres/:id GET :id JSON object the genre
returns a list of all users /users GET none JSON object of all the users
